几年前我在李文威的《代数学方法上》看到偏序集上的 Mobius 反演公式。最近我学习了一种跟枚举倍数有关的技巧，它可以解决 **gcd 卷积**问题。

对于这篇题为 [[Tutorial] GCD Convolution](https://codeforces.com/blog/entry/112346) 的文章，有两个评论引起我的注意

一个是 [Bashca 的评论](https://codeforces.com/blog/entry/112346?#comment-1001131)

> Nice! You can optimize it to $O(n\log\log n)$ using dp sos.
>
> ```cpp
> for (int i=0; p[i]<=n; ++i) {
>   for (int j=n/p[i]; j>=1; --j) {
>     a[j] += a[j * p[i]];	
>     b[j] += b[j * p[i]];
>   }
> }
> 
> for (int i=1; i<=n; ++i) d[i] = a[i] * b[i];
> 
> for (int i=0; p[i]<=n; ++i) {
>   for (int j=1; j * p[i] <= n; ++j) {
>     d[j] -= d[j * p[i]];
>   }
> }
> ```

另一个是 [nor 的评论](https://codeforces.com/blog/entry/112346?#comment-1001686)

> This kind of convolution is very closely related to [Mobius inversion on posets](https://codeforces.com/blog/entry/98413). More specifically, in this case, you look at the zeta transform of the array under the poset formed by the indices according to the divisor lattice. You can generalize this technique to other lattices as well.

于是我想学习一下偏序集上 Mobius 反演公式。此外，这篇题为 [炫酷广义莫比乌斯反演魔术](https://www.luogu.com.cn/article/pli5b78c) 的文章也介绍了偏序集上莫比乌斯反演公式。
